# Customer Support Chatbot using Dialogflow

This project is an AI-powered customer support chatbot built using Dialogflow Essentials.

## Features
- Order tracking using Order ID
- Returns and refund handling
- Billing issue support
- Customer support contact flow
- NLP-based intent classification
- Entity extraction and slot filling
- Multi-turn conversational design

## Tools & Technologies
- Dialogflow Essentials
- Natural Language Processing (NLP)
- Conversational AI
